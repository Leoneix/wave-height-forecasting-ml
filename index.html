Significant Wave Height Forecasting

Comparative evaluation of Linear Regression, Random Forest, XGBoost, and LSTM models for short-term ocean wave prediction.

GitHub LinkedIn

Project Overview

This project investigates short-term forecasting of Significant Wave Height (WVHT) using multivariate buoy data under strict chronological validation.

Methodology

Chronological 80–20 train-test split was used to prevent data leakage.

WVHTt = WVHTt-1

  • Linear Regression
  • Random Forest
  • XGBoost
  • LSTM (24-hour window)

Results

Model R² Score
Linear Regression 0.9599
Random Forest 0.9650
XGBoost 0.9698
LSTM 0.8875
